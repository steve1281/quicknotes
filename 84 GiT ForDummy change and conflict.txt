$ git clone http://consappdev.tbaytel.com/git/DealerPortalMiddleLayer.git

Change a file.
$ git commit -m "added myself to the comments" -a
$ git push

Deal with conflicts.

$ git pull
-- edit with favorite conflict tool --
$ git commit -m "after the merge " -a
$ git push
