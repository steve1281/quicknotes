1. create a new schema

- its under the database, under security, under schema
- you will need an account
- I used mytbaytel as an account when I created the schema mytbaytel

2. Create a blank table

- script the orginal table, rclick script to clipboard (as CREATE)
- paste into a new query eg:

USE [tbaytel]
GO

/****** Object:  Table [dbo].[Customer]    Script Date: 04/10/2012 10:23:51 ******/
SET ANSI_NULLS ON
GO

SET QUOTED_IDENTIFIER ON
GO

CREATE TABLE [dbo].[Customer](
	[CustomerID] [int] IDENTITY(1,1) NOT NULL,
	[UserID] [int] NOT NULL,
	[PortalID] [int] NOT NULL,
	[AddDate] [datetime] NOT NULL,
	[ChangeDate] [datetime] NULL
) ON [PRIMARY]

GO

ALTER TABLE [dbo].[Customer] ADD  CONSTRAINT [DF_Customer_AddDate]  DEFAULT (getdate()) FOR [AddDate]
GO
